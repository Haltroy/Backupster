# Security Policy

## Reporting a Vulnerability
To report a vulnerability, please use [`Vulnerability template`](https://github.com/Haltroy/Backupster/issues/new?assignees=Haltroy&labels=bug%2C+help+wanted%2C+vulnerability&template=vulnerability.md&title=Vulnerability+-+) while creating an issue. 

## Supported Versions

This section currently tells which versions of Backupster is supported.

| Version | Supported          | Available on GitHub | Status         |
| ------- | ------------------ |---------------------|----------------|
| 0.0.1.0 | :white_check_mark: | :white_check_mark:  | **Latest**     |
| 0.0.0.0 | :x:                | :x:                 | End of support |
